About Me

I'm a back-end web developer, currently working for BMW Group developing an interface that allows easy access and editing of several items for post-production vehicle-equipment-combinations, verifies data consistency of said edited items with several sources of truth, and creates reports of aggregated information about a particular set of models, and their possible equipments, with all aforementioned items.


I mainly work with Java/Kotlin, AWS Services such as DynamoDB, Cloudformation, Lambdas and S3, along with a web of other interfacing AWS systems like SNS or SQS, Github Actions for pipelining, and ArgoCD for Kubernetes management.
